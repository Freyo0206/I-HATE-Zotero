Zotero安装（mac版）

本节介绍如何在Mac电脑上安装Zotero。

确保Zotero安装包已下载完成。

  1. 找到安装包的位置，双击触控板或鼠标左键打开。
  2. 点击左侧图标，拖动至右侧文件夹内。
  3. 拖动后稍等片刻，若能在mac启动台能够发现Zotero图标，即安装成功。
  4. 单击图标，点击打开，即可启动Zotero。
