下载失败

Condition

Zotero安装包下载失败。

Cause

网络中断或不稳定。

Remedy

IT Center

  1. 切换WiFi，或使用有线网络、手机热点。
  2. 重新尝试下载。

电脑内存不足

Remedy

Personal Computer

  1. 清理电脑内存，确保可用空间大于安装包大小。
  2. 重新尝试下载。

Zotero安装（win版）

本节介绍如何在Windows系统电脑上安装Zotero。

确保Zotero安装包已下载完成。

  1. 找到安装包的位置，双击触控板或鼠标左键打开。
  2. 点击Next>。
  3. 若采用默认安装路径，则选择Standard后点击Next>，并跳过第4步；若需自定义安装路径，选择Custom后点击Next>。
  4. 点击Browse，选择好安装路径后，点击Next>。
  5. 若需要在桌面创建Zotero图标，则勾选On my Desktop；若需要Zotero图标出现在开始界面菜单，则勾选In my Start Menu Programs folder。完成后点击Next>。
  6. 点击Install，开始安装。
  7. 安装完成后，点击Finish
  8. 若能在桌面或安装路径下找到Zotero图标，即安装成功。
  9. 双击图标，即可启动Zotero。

安装失败（win版）

Condition

在Windows系统电脑上安装Zotero失败。

Cause

电脑内存不足。

Remedy

Personal Computer

  1. 清理电脑内存，确保可用空间大于软件大小。
  2. 重新尝试安装。
Cause

安装包版本与电脑型号不匹配。

Remedy

Personal Computer

  1. 找到此电脑，点击属性，查看电脑位数
  2. 在系统类型中查看电脑位数，检查与安装包版本是否一致。
  3. 如果不一致，则重新下载相应的安装包进行安装，详见"Zotero下载（其他版本）"一节。

无法找到Zotero图标（win版）

Condition

找不到Zotero的图标，无法使用。

Cause

安装时未勾选On my Desktop。

Remedy
  1. 若安装时选择Standard，则打开电脑C盘，打开Program Files (x86)文文件夹，进入Zotero文件夹，找到Zotero图标，双击即可使用。
  2. 若安装时选择Custom，则打开安装时选择的路径。例如，图中自定义安装为"D:\Zotero\"，则打开电脑D盘Zotero文件夹，找到Zotero图标，双击即可使用。
  3. 若需将Zotero图标添加至桌面，则选中Zotero图标后按右键，点击发送到>桌面快捷方式，随后便可在桌面直接打开Zotero。

Zotero安装（mac版）

本节介绍如何在Mac电脑上安装Zotero。

确保Zotero安装包已下载完成。

  1. 找到安装包的位置，双击触控板或鼠标左键打开。
  2. 点击左侧图标，拖动至右侧文件夹内。
  3. 拖动后稍等片刻，若能在mac启动台能够发现Zotero图标，即安装成功。
  4. 单击图标，点击打开，即可启动Zotero。
Related information
  • 安装失败（win版）

安装失败（mac版）

Condition

在Mac电脑上安装Zotero失败。

Cause

电脑内存不足。

Remedy

Personal Computer

  1. 清理电脑内存，确保可用空间大于软件大小。
  2. 重新尝试安装。
Cause

Mac电脑系统权限未打开。

Remedy

Personal Computer

  1. 打开设置
  2. 找到隐私与安全性
  3. 找到安全性，选中App Store与已知开发者
Cause

拖动过程有误。

Remedy

Personal Computer

重新拖动图标，直至出现绿色加号，确保将安装包拖入文件夹内
