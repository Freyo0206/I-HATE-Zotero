安装失败（win版）

Condition

在Windows系统电脑上安装Zotero失败。

Cause

电脑内存不足。

Remedy

Personal Computer

  1. 清理电脑内存，确保可用空间大于软件大小。
  2. 重新尝试安装。

Cause

安装包版本与电脑型号不匹配。

Remedy

Personal Computer

  1. 找到此电脑，点击属性，查看电脑位数
  2. 在系统类型中查看电脑位数，检查与安装包版本是否一致。
  3. 如果不一致，则重新下载相应的安装包进行安装，详见"Zotero下载（其他版本）"一节。
