    • 7.2 同步
      • Zotero同步介绍

        Zotero默认将所有数据存储在本地计算机。但您可于登录后利用 Zotero 的同步功能在任何可上网的计算机访问 Zotero 库。Zotero 同步分为两类：数据同步（Data Syncing）和文件同步（File Syncing）。

      • 7.2.1 数据同步设置

        数据同步可将您储存在本地的文库分类、条目信息、注释、笔记等上传至 Zotero 服务器（除附件外），您可以在任何安装 Zotero 的计算机上使用该数据，并在 zotero.org 上在线查看图书馆。数据同步免费且不受限，您可以在不同步文件的情况下使用。

      • 7.2.2 文件同步设置

        文件同步指同步Zotero中的附件，如pdf文件、caj文件。因数据同步并不同步附加的文件，要同步这些文件，您可使用Zotero Storage或WebDAV以进行文件同步的设置，对数据同步进行补充。

同步过程中的网络连接问题及应对方式

问题描述

同步过程中出现网络连接相关问题。

原因

Zotero 的同步功能依赖于稳定的网络连接。如果网络不稳定，可能会导致同步失败或速度缓慢。

解决方案

尽量在网络良好的环境下进行同步；如果同步出现问题，检查网络连接是否正常。
